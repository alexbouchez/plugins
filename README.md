# Alex Bouchez — Claude Code plugins

Marketplace publique pour mes plugins Claude Code partageables.

## Install

```
/plugin marketplace add alexbouchez/plugins
/plugin install dougs@alexbouchez
```

## Plugins

| Plugin | Repo | Description |
|---|---|---|
| `dougs` | [alexbouchez/dougs](https://github.com/alexbouchez/dougs) | Gestion des brouillons de devis Dougs via fetch direct + cookie session. Brouillon-only. |

> Cette marketplace ne référence que les plugins publics. Mes plugins persos (`linkedin-ghostwriter`, `x-ghostwriter`) vivent dans des repos privés, distribués séparément.
